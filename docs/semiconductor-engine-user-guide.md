# Semiconductor Engine — User Guide
**Reading the AI Regime Lens in under 2 minutes**

---

## Step 1 — Check the AI Regime Label

The AI Regime label (top KPI strip) summarizes the current semiconductor capital flow structure.

| Label | Meaning |
|-------|---------|
| AI-led Broadening | AI leadership confirmed across multiple semiconductor segments |
| Narrow AI Leadership | AI Compute is leading, other segments are lagging |
| Capital Rotation | Capital is moving from AI infrastructure toward equipment and foundry |
| Broad Recovery | Participation is broad across all semiconductor segments |
| Semiconductor Contraction | Structural weakness confirmed across all segments |

---

## Step 2 — Read the Relative Spread Chart (CYCLE VIEW [1])

This chart shows which groups are stronger or weaker than SOXX.

- **Line above zero** → that bucket is outperforming SOXX → supporting the structure
- **Line below zero** → that bucket is lagging SOXX → weakening the structure
- **Zero line** = SOXX

Read left-to-right: is the gap widening or narrowing?

---

## Step 3 — Check the Rebased Bucket Flow (CYCLE VIEW [2])

This chart compares bucket movement from the same starting point.

All series start at zero. Watch:
- Which group pulled ahead first
- Which groups followed
- How much separation exists between AI Compute and the others

Separation = narrow leadership. All together = broad participation.

---

## Step 4 — Check the Capital Flow Stage (CYCLE VIEW [3])

This timeline shows how far AI capital has confirmed across the semiconductor value chain.

Flow order: **AI Compute → Memory → Foundry → Equipment → Broad**

- **Confirmed** (green) = clearly outperforming SOXX
- **Partial** (blue) = moderate outperformance
- **Mixed** (yellow) = near SOXX baseline
- **Lagging / Weak** (orange/red) = underperforming SOXX

Healthy cycle: stages turn green sequentially from left to right.

---

## Step 5 — Read SOXL Sensitivity (Right Panel ⑤)

SOXL sensitivity shows how concentrated or broad SOXX support may affect leveraged ETF behavior.

- **High** = narrow leadership or contraction — leverage amplifies risk
- **Medium** = rotation or partial recovery — uneven
- **Low–Medium** = broadly supported structure

This is structural context, not a position sizing instruction.

---

## Step 6 — Check Data Status (Footer)

Always check whether the data is:

| Status | Meaning |
|--------|---------|
| LIVE | Data is from a real-time market API |
| SNAPSHOT | Data is from the most recent cached file |
| FALLBACK | Static fallback is displayed |
| UNAVAILABLE | No data could be loaded |

Last Updated shows the date of the most recent data.

---

## Summary Reading Flow

```
AI Regime Label → Relative Spread → Rebased Flow → Capital Flow Stage → SOXL Sensitivity → Data Status
```

Total reading time: under 2 minutes.
