# Prompt Docs

This folder is a snapshot of where prompts are used in MarketFlow.

## Scope

- registry-backed prompts
- shared prompt libraries
- inline prompt strings
- reference-only assets

## Files

- [inventory.md](./inventory.md): prompt asset inventory
- [runtime-map.md](./runtime-map.md): actual code callsites
- [versioning.md](./versioning.md): registry and versioning rules

## How to read

1. Start with registry-backed prompts.
2. Then check shared prompt libraries.
3. Finish with inline prompts and reference-only assets.

Only prompt text that is sent to an LLM is included here.
UI labels, output text, and non-prompt strings are excluded.
