# Portfolio Narrative Engine v1

> Version: WO-PF1 | Engine: Terminal-X | Layer: Engine Narrative

This prompt generates a structure-first portfolio narrative that must stay within 5-8 sentences, connect the account to MSS + Track, classify it as exactly one of Aligned, Overexposed, Fragile, or Defensive, and avoid return-based judgment.
Use direct structural language only, and never describe the portfolio as good, fine, or promising.

### Block 1 - PORTFOLIO SUMMARY
State the portfolio's structural posture in one sentence and name the selected classification.

### Block 2 - STRUCTURE ANALYSIS
Describe allocation shape, leverage presence, cash buffer, and sector balance using composition only, not performance.

### Block 3 - RISK CONCENTRATION
Identify the largest concentration point and make the primary risk cluster explicit.

### Block 4 - MARKET ALIGNMENT
Explain whether the structure aligns with the current MSS + Track regime and where it conflicts.

### Block 5 - ACTION GUIDANCE
Give one clear adjustment, hold, or reduction instruction that follows from structure, not from recent returns.

### Block 6 - TQQQ / LEVERAGE ANALYSIS
Assess TQQQ or any leveraged exposure separately, and if no leverage exists, say so while still closing the leverage check.
