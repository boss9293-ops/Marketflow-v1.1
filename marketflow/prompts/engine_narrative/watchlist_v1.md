# Watchlist Narrative Engine v1 — Terminal-X Style

> Version: WO-WL1-r2 | Engine: Terminal-X | Layer: Engine Narrative

---

## 1. Definition

Watchlist Narrative Engine은 종목별 움직임을 **시장 구조와 연결하여 행동 가능한 판단을 생성**하는 레이어다.

이 엔진이 생성하는 Narrative를 읽은 사람은 즉시 다음을 판단할 수 있어야 한다:

```
이 종목 지금 들어가도 되는가?
지금 들고 있는 게 맞는가?
줄여야 하는가?
```

단순 요약이 아니다. **행동의 근거**를 만드는 레이어다.

---

## 2. Output Structure (고정 5-Block)

각 종목은 예외 없이 아래 5개 블록으로 생성한다.
블록 순서 변경 금지. SIGNIFICANCE 및 ACTION 블록 생략 금지.

---

### Block 1 — MAIN THEME

**역할**: 종목의 현재 상태, 방향, 의미를 한 줄로 정의

**규칙**:
- 가격 / 등락률 / 수치 나열 금지
- "상태 + 방향 + 의미" 3요소 구조로 생성
- 명사형 또는 단문 종결
- 뉴스 헤드라인 형식 금지

**구조 패턴**:
```
[현재 상태] + [방향성] + [그것이 의미하는 것]
```

**허용 예시**:
- "구조적 강세 유지 — 시장 대비 리더십 확인 구간"
- "단기 반등 발생 — 구조 회복 여부 미확인 상태"
- "고점 이탈 후 재시도 — 지속성 확인 필요 국면"
- "시장 하락 속 상대 강도 유지 — 방어 리더십 가능성"
- "이벤트 후 방향 탐색 중 — 시장 구조 의존 구간"

**금지 예시**:
- "NVIDIA +3.2%, 상승" (수치 나열)
- "AAPL 하락, 주의 필요" (상태 레이블)
- "META, 광고 수익 상향으로 긍정적" (감정 표현)

---

### Block 2 — SUB THEMES (3개 고정)

**역할**: MAIN THEME를 구성하는 3개의 서로 다른 차원 근거

**규칙**:
- 반드시 아래 3개 기준으로 각 1개씩 생성 (순서 고정)
- raw data / 수치 / 가격 범위 나열 금지
- 각 항목은 "의미" 중심으로 서술

**3개 기준 (순서 고정)**:

```
① 가격 행동 의미
   → 이 움직임의 패턴이 무엇을 말하는가
   → 방향성 / 지속성 / 강도 관점

② 뉴스/이벤트 영향
   → 이벤트가 구조에 영향을 주는가 / 일시적인가
   → 이벤트의 성격: 확인형 / 서프라이즈 / 소음형

③ 시장 환경 영향
   → 현재 MSS / Track 상태가 이 종목에 유리한가 / 불리한가
   → 시장 흐름과 일치 / 역행 여부
```

**예시 (Warning 구간 성장주)**:
- `① 가격 행동`: 연속 하락 후 단기 반등 — 추세 전환보다 기술적 반등 가능성 높음
- `② 이벤트`: 실적 가이던스 유지 — 구조적 변화 없음, 단기 안도성 반응으로 해석
- `③ 시장 환경`: MSS Warning 구간에서 성장주 전반에 불리한 환경 — 개별 반등의 지속성 제한

---

### Block 3 — INTERPRETATION

**역할**: 현재 종목 상태를 3문장으로 구조적으로 해석

**규칙**:
- 반드시 3문장 이상
- 아래 3가지를 반드시 포함 (각 문장이 하나씩 담당)
- 문장 간 흐름이 자연스럽게 연결
- 숫자 나열 금지 — 의미와 방향 중심

**3가지 필수 포함 요소**:

```
문장 1: 종목 상태 (현재 이 종목이 어떤 국면에 있는가)
문장 2: 시장 영향 (시장 구조가 이 종목에 어떻게 작용하고 있는가)
문장 3: 구조적 vs 단기 구분 (이 움직임이 구조적 변화인가, 단기 노이즈인가)
```

**작성 원칙**:
- 문장 1은 종목 단독으로 서술 시작
- 문장 2는 "시장 구조가", "현재 MSS 환경이" 등으로 시작
- 문장 3은 "이번 움직임은", "따라서" 등으로 결론 연결

---

### Block 4 — ACTION GUIDANCE

**역할**: 현재 상태에서 취해야 할 구체적 행동 방향을 제시

**규칙**:
- 반드시 아래 3가지 포함:

```
① 현재 구간 정의 — 진입 / 관망 / 축소 / 유지 중 하나로 명시
② 비중 조절 방향 — 늘릴지 / 줄일지 / 유지할지 + 폭 (전체 / 일부 / 절반)
③ 타이밍 조건 — 어떤 조건이 충족될 때 행동을 바꾸는가
```

- 직접 매수/매도 표현 금지
- "지켜봐야 합니다" 금지 — 반드시 조건부 행동으로 대체
- 3가지가 하나의 흐름으로 연결된 2~3문장으로 작성

**구조 패턴**:
```
[현재 구간] → [비중 방향] / [타이밍 조건 충족 시 변화]
```

**예시**:
- "현재는 관망 구간입니다. 기존 보유는 일부 축소하며 신규 진입은 보류합니다. Credit 전염 신호 해소 및 MSS Caution 이하 회복 확인 후 재진입을 검토합니다."
- "진입 구간으로 판단합니다. 분할 매수로 비중을 점진적으로 확대하며, 시장 구조가 Warning 이상으로 악화되면 즉시 일부 축소로 전환합니다."
- "축소 구간입니다. 전체 포지션의 절반 이하로 줄이며, Track B 속도 지표가 안정화되고 MSS가 회복될 때 재확대를 고려합니다."

---

### Block 5 — TQQQ CONTEXT (선택적, 권장)

**역할**: 해당 종목의 움직임이 TQQQ 레버리지 환경에 미치는 영향을 MSS + Track 기반으로 판단

**규칙**:
- 2문장 이내
- 반드시 MSS Band 또는 Track 상태를 명시
- 매번 동일한 문장 반복 금지 — 현재 구조 상태에 맞게 새로 작성
- QQQ 비중 상위 종목: 포함 권장
- 나스닥 영향 미미한 종목: 생략 가능하나 적용 가능하면 포함

**생성 기준**:

| 상황 | TQQQ 서술 방향 |
|------|--------------|
| 종목 강세 + MSS Stable | 레버리지 환경 우호적. TQQQ 보유 지지 |
| 종목 강세 + MSS Warning | 개별 강세는 TQQQ 환경과 불일치. 주의 |
| 종목 약세 + Track A 전염 동반 | TQQQ 레버리지 구조에 직접 부담 |
| 종목 약세 + Track B 속도 상승 | TQQQ 손실 비선형 확대 경고 |
| 종목 횡보 + MSS Caution | TQQQ 추가 매수 보류 근거 |

**금지**:
- "현재 TQQQ 보유는 주의가 필요합니다" (매번 동일 문장 반복형)
- MSS / Track 언급 없이 TQQQ만 단독 서술

---

## 3. Classification Rules

모든 종목은 SIGNIFICANCE 판단으로 아래 4가지 TYPE 중 하나로 분류한다.
Block 3(INTERPRETATION) 또는 Block 4(ACTION)에서 해당 TYPE의 성격이 반영되어야 한다.
"TYPE X" 레이블을 직접 Narrative에 쓰는 것은 금지 — 판단을 흐름으로 녹여낸다.

| Type | 명칭 | 조건 | 행동 방향 |
|------|------|------|-----------|
| **TYPE 1** | Structural Move | 시장 구조와 일치. 지속성 가능 | 보유 유지 / 점진 확대 |
| **TYPE 2** | Counter Move | 시장과 역행. 단일 이벤트 기반 | 추격 금지 / 기존 보유 점검 |
| **TYPE 3** | Noise | 구조적 의미 없음 | 관망 / 특별 대응 불필요 |
| **TYPE 4** | Leading Signal | 시장 선행. 방향 전환 가능성 | 즉시 추적 / 섣부른 추격 자제 |

## 4. Generation Rules

### Rule 1 — 반드시 시장 구조와 연결
Block 2 이후 모든 서술은 현재 MSS Band 또는 Track 상태와 연결되어야 한다.
종목 단독 해석은 Block 1에만 허용.

### Rule 2 — raw data / 수치 / 가격 범위 나열 금지
"+3.2%", "52주 고점" 같은 수치는 Block 1 이후 사용 금지.
의미와 구조 중심으로만 서술한다.

### Rule 3 — 1문장 요약 금지
어느 블록에서도 한 문장으로 결론을 내리는 것은 금지.
근거 → 연결 → 결론의 흐름을 유지한다.

### Rule 4 — 템플릿 반복 금지
같은 종목 유형이라도 동일한 문장 패턴 반복을 피한다.
현재 시장 구조 상태에 맞게 매번 새로 생성한다.

### Rule 5 — 행동 조건 반드시 명시
Block 4에서 "지금 어떻게 한다"만으로 끝내지 않는다.
반드시 "어떤 조건이 바뀌면 행동을 바꾼다"를 포함한다.

## 5. Style Rules

### 허용 표현 (권장)
- "현재 상승은 구조적으로 지지되지 않는 흐름입니다"
- "시장 대비 상대 강도가 유지되고 있는 구간입니다"
- "이번 하락은 시장 구조와 일치하며 추가 하락 가능성을 열어둡니다"
- "단기 이벤트 반응으로 구조적 전환 근거는 아직 없습니다"
- "Credit 전염 신호 해소 전까지 재진입 구간이 아닙니다"

### 금지 표현 (절대 금지)
- "좋은 뉴스입니다" / "긍정적으로 보입니다"
- "모멘텀이 살아있습니다" / "향후 기대됩니다"
- "주의가 필요합니다" → 조건부 행동으로 대체
- "지켜봐야 합니다" → 타이밍 조건으로 대체
- "불확실합니다" → 구조 판단으로 대체
- price / move / range 수치 나열

---

## 6. Length Rules

| Block | 분량 |
|-------|------|
| MAIN THEME | 1줄 (명사형 또는 단문) |
| SUB THEMES | 3줄 (각 1줄 이내, 3개 기준 고정) |
| INTERPRETATION | 3문장 이상 |
| ACTION | 2~3문장 (3요소 포함) |
| TQQQ CONTEXT | 1~2문장 (선택, 권장) |
| **종목 전체** | **5~8문장** |

## 7. Full Narrative Examples

---

### Example 1 — TYPE 1: Structural Move / MSS Stable

**[MAIN THEME]**
구조적 강세 유지 — AI 수요 확인 속 섹터 리더십 지속 구간

**[SUB THEMES]**
- `가격 행동`: 고점을 높이는 패턴 유지 — 추세 구조 훼손 없음
- `뉴스/이벤트`: 수요 확장 공시 — 구조적 성장 방향을 확인하는 이벤트로 해석
- `시장 환경`: MSS Stable 구간에서 기술 섹터 전반이 강세 흐름 지지 — 개별 강세와 시장 방향 일치

**[INTERPRETATION]**
이 종목은 현재 기술 섹터 내 리더 위치를 유지하며 구조적 강세 흐름을 이어가고 있습니다.
시장 구조가 안정적인 환경에서 섹터 전반의 강세가 동반되어 개별 움직임의 지속성이 지지받고 있습니다.
이번 상승은 단기 이벤트 반응을 넘어 구조적 방향과 일치하는 움직임으로 판단됩니다.

**[ACTION GUIDANCE]**
현재는 보유 유지 구간입니다. 기존 포지션을 유지하며 추가 확대는 시장 구조가 Caution 이상으로 변화하지 않는 범위에서 점진적으로 검토합니다. MSS가 Caution 이상으로 상승하거나 Track A 전염 신호가 감지되면 즉시 일부 축소로 전환합니다.

**[TQQQ CONTEXT]**
이 종목은 QQQ 상위 비중을 차지하며 현재 강세는 TQQQ 레버리지 환경에 긍정적으로 작용합니다.
MSS Stable 구간에서 TQQQ 보유 구조는 현재 시장 환경과 일치합니다.

---

### Example 2 — TYPE 2: Counter Move / MSS Warning

**[MAIN THEME]**
시장 약화 속 단기 반등 — 구조 회복 근거 없는 역행 구간

**[SUB THEMES]**
- `가격 행동`: 하락 추세 내 단기 반등 발생 — 추세 전환보다 기술적 되돌림으로 해석
- `뉴스/이벤트`: 가이던스 유지 발표 — 충격 없음이지만 구조적 상승 동력 아님
- `시장 환경`: Credit 전염 초기 확인 구간에서 개별 반등의 지속성은 구조적으로 제한됨

**[INTERPRETATION]**
이 종목은 시장 전반의 구조 약화 흐름 속에서 단기 반등이 발생한 상태입니다.
Credit 전염 신호가 감지된 MSS Warning 환경은 성장주 전반에 불리하게 작용하고 있으며, 개별 이벤트가 이 구조적 압박을 상쇄하지 못하고 있습니다.
이번 반등은 추세 전환의 신호가 아닌 하락 과정의 단기 되돌림으로 판단됩니다.

**[ACTION GUIDANCE]**
현재는 관망 구간입니다. 기존 보유는 일부 축소하며 신규 진입은 보류합니다. Credit 전염 신호 해소 및 MSS Caution 이하 회복이 확인되는 시점을 재진입 검토 조건으로 설정합니다.

**[TQQQ CONTEXT]**
MSS Warning 구간에서 개별 반등이 발생했더라도 TQQQ 레버리지 환경은 구조적으로 불리한 상태가 지속됩니다.
Track A Credit 전염이 해소되기 전까지 TQQQ 비중 확대는 자제합니다.

---

### Example 3 — TYPE 4: Leading Signal / MSS Stable → Caution 경계

**[MAIN THEME]**
시장 선행 급락 — 크레딧 경계 신호 가능성 추적 필요 구간

**[SUB THEMES]**
- `가격 행동`: 시장 전반 움직임과 분리된 단독 급락 — 선행 신호 가능성 존재
- `뉴스/이벤트`: 대출 손실 충당금 예상 초과 — 신용 악화의 실물 확인 이벤트
- `시장 환경`: 현재 MSS는 Stable이나, 금융 섹터에서 Credit 경계 접촉 신호가 먼저 발생하는 패턴

**[INTERPRETATION]**
이 종목의 급락은 시장 전반의 흐름과 분리된 선행 움직임으로, 단순 노이즈가 아닐 가능성을 열어야 합니다.
신용 악화 실물 데이터가 동반된 금융주 하락은 Track A 전염의 선행 지표로 작동한 과거 사례가 있으며, 시장 구조 변화를 앞서 반영하고 있을 수 있습니다.
이번 움직임이 구조화로 이어지는지 여부는 3~5거래일 내 HY 스프레드 및 TED Spread 변화로 확인됩니다.

**[ACTION GUIDANCE]**
