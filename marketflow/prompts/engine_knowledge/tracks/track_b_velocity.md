# Track B — Velocity Knowledge Layer

> Version: WO-TB1 | Engine: Terminal-X | Layer: Engine Knowledge

---

## 1. Definition

Track B는 시장 리스크의 **속도(Speed)와 가속도(Acceleration)**를 측정하는 레이어다.

MSS가 "시장이 얼마나 건강한가"를 측정하고,
Transmission이 "리스크가 어디로 이동하는가"를 추적한다면,
Track B는 **"그 이동이 얼마나 빠르게 일어나고 있는가"**를 측정한다.

### 핵심 명제
- 같은 수준의 리스크라도, **속도가 빠르면 위험은 기하급수적으로 증가**한다
- Velocity는 "시장 참여자의 대응 가능 시간"을 줄인다
- 속도가 빠를수록 레버리지 자산의 손실은 비선형적으로 확대된다

### MSS와의 차이
| 구분 | MSS | Track B (Velocity) |
|------|-----|-------------------|
| 측정 대상 | 시장 구조 건강도 | 리스크 변화 속도 |
| 반응 속도 | 상대적 후행 | 선행 신호 가능 |
| 단독 신뢰도 | 높음 | 낮음 (노이즈 가능) |
| 결합 효과 | Velocity와 결합 시 강화 | MSS/Track A와 결합 시 유효 |

**원칙**: Track B는 단독으로 사용하지 않는다. 반드시 MSS 또는 Track A와 함께 해석한다.

## 2. Velocity Levels

### L0 — Normal
- **의미**: 시장 변화 속도가 정상 범위 내에 있음
- **위험 수준**: 없음
- **해석 키워드**: 안정적 추세, 정상 변동성, 대응 시간 충분
- **특징**: VIX 정상, 단기 모멘텀 일관, 급격한 구조 변화 없음

### L1 — Early Acceleration
- **의미**: 속도가 정상 범위를 벗어나기 시작함. 가속의 초기 신호
- **위험 수준**: 낮음 (감시 개시)
- **해석 키워드**: 초기 가속, 변동성 확대 조짐, 주의 구간 진입
- **특징**: VIX 상승 시작, 단기 하락 속도 평균 초과, 아직 패닉 없음

### L2 — Elevated Velocity
- **의미**: 하락 속도가 유의미하게 빨라짐. 구조 변화 가능성 존재
- **위험 수준**: 중간 (대응 시간 감소)
- **해석 키워드**: 속도 상승 확인, 시장 반응 지연 위험, 레버리지 부담 증가
- **특징**: 수일 내 급락, VIX 급등, 옵션 프리미엄 확대

### L3 — High Velocity
- **의미**: 시장이 빠르게 재편되고 있음. 정상적 대응 시간 부족
- **위험 수준**: 높음 (즉각 대응 필요)
- **해석 키워드**: 빠른 재편, 대응 시간 부족, 레버리지 위험 최고조
- **특징**: 연속 급락, 일간 -3% 이상 반복, VIX 30+ 진입, 공포 매도 시작

### L4 — Crisis Velocity
- **의미**: 시장 붕괴에 가까운 속도로 하락 중. 시스템적 패닉
- **위험 수준**: 극단 (생존 모드)
- **해석 키워드**: 패닉 속도, 유동성 소멸, 레버리지 자산 강제 청산
- **특징**: 연속 -5% 이상, VIX 40+, 마진콜 연쇄, 상관관계 1 수렴

## 3. Interaction Rules

### Rule 1 — Track B 단독 해석 금지
- L1~L2 단독 신호는 노이즈일 수 있음
- 반드시 MSS 레벨 또는 Track A 방향과 함께 해석
- Velocity만 높고 MSS가 정상이면 → 일시적 변동 가능성 우선 고려

### Rule 2 — MSS와 결합 시
| MSS 상태 | Velocity | 해석 |
|----------|----------|------|
| Healthy (110+) | L1~L2 | 일시적 조정 가능성. 모니터링 |
| Neutral (100~110) | L2~L3 | 주의. 속도가 구조를 앞서고 있음 |
| Risk (92~100) | L2~L3 | 위험 증폭. 빠른 포지션 조정 필요 |
| High Risk (<92) | L3~L4 | 즉각 방어. MSS 악화 + 속도 위험 동시 |
| Crisis (<84) | L3~L4 | 생존 모드. 전량 방어 전환 |

### Rule 3 — Transmission과 결합 시 (최상위 위험)
- Transmission Active 이상 + Velocity L3 이상 = **최상위 위험 조합**
- 이 조합에서는 MSS가 양호해도 즉각 방어 행동
- 2020-03, 2008-09 붕괴는 모두 이 조합에 해당

```
Transmission Active + Velocity L3 = A4/A5 즉시 발동
```

### Rule 4 — Track A와의 관계
- Track A: 리스크의 "방향"과 "위치" 측정
- Track B: 리스크의 "속도"와 "가속도" 측정
- Track A가 하락 방향을 확인하고, Track B가 속도를 확인하면 → 신호 신뢰도 상승

## 4. Action Mapping

| Level | Action Code | 행동 지침 |
|-------|-------------|-----------|
| L0 Normal | **A1** | 정상 포지션 유지. TQQQ 보유 가능 |
| L1 Early Acceleration | **A2** | 감시 강화. 추가 매수 보류. 소폭 축소 검토 |
| L2 Elevated Velocity | **A3** | 속도 위험 확인. TQQQ 50% 이하. 현금 확보 시작 |
| L3 High Velocity | **A4** | 즉각 대응. TQQQ 대부분 청산. 방어 포지션 구축 |
| L4 Crisis Velocity | **A5** | 생존 모드. 전량 현금/달러/단기채 전환. 신규 진입 금지 |

**결합 조건에 따른 Action 상향 조정:**
- MSS Risk + Velocity L2 → A3 → A4로 상향
- Transmission Active + Velocity L3 → 즉시 A5 발동 검토

## 5. TQQQ Implication

Track B는 TQQQ 보유자에게 가장 직접적인 위험 신호다.
TQQQ는 **일간 변동성이 기하급수적으로 증폭**되기 때문에,
속도가 빠른 환경에서는 손실이 비선형적으로 확대된다.

### TQQQ 리스크 매트릭스

| Level | TQQQ 리스크 | 권고 |
|-------|------------|------|
| L0 | 정상 변동성 | 보유 가능 |
| L1 | 주의 구간 | 추가 매수 보류 |
| L2 | 위험 증가 | 절반 이하 축소 권고 |
| L3 | **고위험** | 대부분 청산. 재진입 대기 |
| L4 | **극단 위험** | 전량 청산 / 생존 모드 |

### L3 이상 핵심 원칙
> "속도가 빠른 환경에서 TQQQ를 보유하는 것은
> 브레이크 없는 차를 내리막길에서 운전하는 것과 같다.
> L3 이상 감지 시 대응 시간은 1~2거래일 이내다."

### Volatility Decay 경고
- TQQQ는 변동성이 클수록 장기 보유 시 손실 (Volatility Decay)
- L2 이상 지속 환경: 매일 보유 비용이 손실로 누적됨
- 속도 환경이 정상화되기 전까지 재진입 금지

## 6. Narrative Rules

### 작성 원칙
1. **속도 중심 서술**: "하락했다" 금지 → "빠르게 재편되고 있다" 형식
2. **시간 부족 개념 포함**: "대응 시간이 줄어들고 있다" 반드시 포함
3. **가속도 표현**: "속도가 높아지고 있다" → "점점 빨라지고 있다" 표현 사용
4. **레버리지 연결**: L2 이상 시 TQQQ 위험을 Narrative에 명시

### Narrative 금지 패턴
- "시장이 하락했습니다" (속도 개념 없음)
- "Track B Level 3입니다" (숫자 나열 금지)
- "변동성이 높습니다" (속도 방향성 없는 서술)

## 7. Narrative Examples

### Example 1 — L0 Normal
> "시장 변화 속도는 정상 범위 내에 있습니다.
> 리스크 이동이 빠르지 않으며 충분한 대응 시간이 확보되어 있습니다.
> TQQQ 포함 포지션을 정상 운영합니다."

### Example 2 — L2 Elevated Velocity
> "시장 하락 속도가 유의미하게 빨라지고 있습니다.
> 수일 내 급격한 구조 변화 가능성이 있으며, 대응 가능 시간이 줄어들고 있습니다.
> TQQQ 포지션을 절반 이하로 축소하고, 속도가 더 빨라지는지 집중 감시합니다."

### Example 3 — L3 High Velocity
> "시장이 빠르게 재편되고 있습니다.
> 연속적인 급락이 이어지며 정상적인 대응 시간이 부족한 상황입니다.
> TQQQ를 포함한 레버리지 포지션은 즉각 대부분 청산이 필요합니다.
> 속도가 이 수준에서는 하루 이틀의 망설임이 큰 손실로 이어질 수 있습니다."

### Example 4 — L4 Crisis Velocity + Transmission Systemic
> "시장이 붕괴에 가까운 속도로 하락하고 있으며,
> 동시에 Credit → Funding → Liquidity 전 경로에 걸쳐 리스크가 전이 중입니다.
> 이는 2008, 2020-03과 동일한 속도 + 전이 구조입니다.
> 신규 진입을 즉시 중단하고, 모든 레버리지 자산을 청산하여 현금으로 전환합니다.
> 대응 시간이 없습니다."
