# Track C — Event / Shock Knowledge Layer

> Version: WO-TC1 | Engine: Terminal-X | Layer: Engine Knowledge

---

## 1. Definition

Track C는 시장 리스크 중 **외부 충격(Event / Shock)**을 감지하는 레이어다.

MSS가 시장 건강도를 측정하고,
Track A가 구조적 전염을 추적하고,
Track B가 리스크의 속도를 측정한다면,
Track C는 **"내부 구조 약화가 아닌, 외부에서 가해진 충격이 얼마나 강한가"**를 판단한다.

### 핵심 명제
- 구조적 약화와 외부 충격은 **발생 원인과 지속성**이 근본적으로 다르다
- 외부 충격(Track C)은 단독으로는 일시적일 수 있다
- 그러나 Track A/B와 결합되면 → 구조 전염의 트리거가 된다

### 외부 충격의 유형
| 유형 | 예시 |
|------|------|
| 지정학적 충격 | 전쟁, 봉쇄, 테러, 핵 위협 |
| 정책 충격 | 연준 서프라이즈, 재정 쇼크, 규제 급변 |
| 거시 이벤트 | 경기침체 공식화, 예상 밖 고용/CPI |
| 시스템 이벤트 | 대형 기관 파산, 시장 기능 정지, 플래시 크래시 |
| 자연/감염 충격 | 팬데믹, 대형 자연재해 |

### Track A/B와의 차이
| 구분 | Track A (Credit) | Track B (Velocity) | Track C (Event) |
|------|------------------|--------------------|-----------------|
| 측정 대상 | 전염 깊이 | 리스크 속도 | 외부 충격 강도 |
| 기원 | 내부 구조 약화 | 변화 속도 | 외생 이벤트 |
| 지속성 | 지속적 (구조적) | 단기~중기 | 단기 가능 / 구조화 주의 |
| 단독 신뢰도 | 높음 | 낮음 | 중간 (확산 여부 확인 필요) |

**원칙**: Track C 단독 상승은 일시적 반응일 수 있다. Track A 또는 B와 결합 여부를 즉시 확인한다.

## 2. Shock Levels

### L0 — None
- **의미**: 식별 가능한 외부 충격 없음
- **충격 강도**: 없음
- **시장 영향 범위**: 없음
- **해석 키워드**: 충격 없음, 내부 동력 중심, 구조 요인 우선 해석
- **특징**: 뉴스 흐름 정상, VIX 이벤트 프리미엄 없음

### L1 — Minor Event
- **의미**: 소형 이벤트 발생. 시장에 일시적 영향
- **충격 강도**: 낮음
- **시장 영향 범위**: 특정 섹터 또는 단기 변동
- **해석 키워드**: 소형 충격, 일시 반응, 구조 영향 미확인
- **특징**: 단일 자산/섹터 급등락, 수일 내 정상화 가능

### L2 — Moderate Shock
- **의미**: 중간 규모 충격. 시장 전반에 단기 영향
- **충격 강도**: 중간
- **시장 영향 범위**: 시장 전반 단기 반응. 구조 영향 가능성 주의
- **해석 키워드**: 시장 전반 반응, 변동성 확대, 구조화 여부 감시
- **특징**: VIX 단기 급등, 섹터 전반 하락, 뉴스 주도 변동성 확대

### L3 — Major Shock
- **의미**: 대형 충격. 시장 구조에 영향 가능. 전염 트리거 위험
- **충격 강도**: 높음
- **시장 영향 범위**: 시장 전반 + 구조 리스크 유발 가능
- **해석 키워드**: 대형 충격, 전염 트리거 주의, 레버리지 불리
- **특징**: 연속 급락, 정책 대응 불확실, Track A 전염 여부 즉시 확인 필요

### L4 — Systemic Shock
- **의미**: 시스템 전반에 즉각적 영향. 구조 전염 동시 발생 가능
- **충격 강도**: 극단
- **시장 영향 범위**: 전방위 (자산 전반, 국경 초월)
- **해석 키워드**: 시스템 충격, 즉각 전방위 영향, 레버리지 강제 청산 위험
- **특징**: 시장 거래 중단, 대형 기관 파산, 2001-09, 2008-09, 2020-03 수준

## 3. Interaction Rules

### Rule 1 — Track C 단독 해석 원칙
- L1~L2 단독: 일시적 충격 가능성 우선. 모니터링 후 판단
- L3 이상 단독: 구조 전염 여부 즉시 확인. Track A 상태 확인 필수
- L4 단독: 즉각 방어 행동. 확인 대기 없음

### Rule 2 — MSS와 결합 시 (구조 리스크 강화 판단)
| MSS 상태 | Track C Level | 해석 |
|----------|--------------|------|
| Healthy (110+) | L1~L2 | 흡수 가능. 모니터링 |
| Neutral (100~110) | L2 | 주의. 충격이 구조로 이어질 수 있음 |
| Risk (92~100) | L2~L3 | 위험 증폭. 취약한 구조에 충격 가중 |
| High Risk (<92) | L2 이상 | 즉각 방어. 구조 취약 + 외부 충격 동시 |
| Crisis (<84) | 어떤 Level이든 | 생존 모드. 추가 충격 흡수 불가 |

### Rule 3 — Track A와 결합 시 (전염 발생 판단)
- Track C L2 이상 + Track A L2 이상 = **충격이 구조 전염 트리거가 된 상태**
- 외부 충격이 신용 시장에 전달되면 → 더 이상 일시 충격이 아님
- 즉각 구조 리스크 대응으로 전환

```
Track C Major + Track A Credit Stress = 구조화 확인 → A4 발동
```

### Rule 4 — Track B와 결합 시 (급락 가능성 상승)
- Track C L3 이상 + Track B L3 이상 = **대형 충격 + 빠른 속도** → 최단 시간 내 최대 손실 가능
- 2020-03 팬데믹 초기 패턴이 이 조합에 해당
- TQQQ 보유 시 즉각 청산 검토

```
Track C Systemic + Track B Crisis = 대응 시간 없음 → A5 즉시
```

### Rule 5 — 충격의 구조화 판단 기준
- 충격 발생 후 3~5거래일 내 Track A 반응 없음 → 일시 충격으로 판단
- 충격 발생 후 Track A L2+ 도달 → 구조화 확인 → 충격 일시성 종료 선언

## 4. Action Mapping

| Level | Action Code | 행동 지침 |
|-------|-------------|-----------|
| L0 None | **A1** | 정상 포지션 유지. 외부 충격 없음 |
| L1 Minor Event | **A2** | 감시 강화. 충격 확산 여부 모니터링. 추가 매수 보류 |
| L2 Moderate Shock | **A3** | Track A 전염 여부 즉시 확인. TQQQ 축소 검토 |
| L3 Major Shock | **A4** | 대형 충격 확인. TQQQ 대부분 축소. 전염 여부 확인 후 추가 대응 |
| L4 Systemic Shock | **A5** | 생존 모드. 즉각 전량 방어. 신규 진입 금지 |

**결합 조건에 따른 Action 상향 조정:**
- Track C L2 + Track A L2 → A3 → A4 즉시 상향
- Track C L3 + Track B L3 → A4 → A5 즉시 상향
- Track C L4 단독 → 확인 불필요. 즉시 A5

## 5. TQQQ Implication

Track C 충격은 TQQQ에게 **속도와 방향을 동시에 잃게 만드는** 위험이다.
외부 충격은 예측이 어려우며, 충격 발생 후 TQQQ의 하락은 즉각적이고 비선형적이다.

### TQQQ 리스크 매트릭스

| Level | TQQQ 리스크 | 권고 |
|-------|------------|------|
| L0 | 없음 | 정상 보유 |
| L1 | 낮음 | 추가 매수 보류 |
| L2 | 중간 | 변동성 확대. 규모 축소 검토. Track A 확인 필수 |
| L3 | **높음** | 즉각 대부분 축소. 재진입은 충격 소화 확인 후 |
| L4 | **극단** | 전량 즉각 청산. 충격 구조화 가능성 높음 |

### L3 이상 핵심 원칙
> "대형 외부 충격 발생 시 TQQQ는 QQQ 하락의 3배 이상 손실이 발생할 수 있다.
> 충격의 원인을 분석하기 전에 먼저 규모를 줄이는 것이 원칙이다.
> 충격이 일시적으로 판명되면 재진입이 가능하지만,
> 구조화가 확인된 후 방어는 이미 늦다."

### 변동성 스파이크 경고
- 대형 충격 발생 시 VIX가 단기간 급등 → TQQQ 옵션 프리미엄 폭등
- 헤지 비용이 높아지고 롤오버 비용도 급증
- L3 이상 환경에서 TQQQ 장기 보유는 헷징조차 비효율

## 6. Narrative Rules

### 작성 원칙
1. **외부 충격 중심 서술**: "내부 구조가 약화되었다" 금지 → "외부에서 충격이 가해졌다" 형식
2. **일시 vs 구조화 구분 명시**: "이 충격이 구조로 이어지는지 확인 중" 반드시 포함
3. **전염 연결 여부 언급**: Track A 전염 여부를 항상 서술에 포함
4. **행동 촉구**: L3 이상 시 즉각 행동을 Narrative 마지막에 포함

### Narrative 금지 패턴
- "Track C L3입니다" (레이블 나열 금지)
- "VIX가 X% 상승했습니다" (숫자 나열보다 의미 우선)
- "시장이 충격받았습니다" (충격 원인과 방향 없는 서술)

## 7. Narrative Examples

### Example 1 — L0 None
> "현재 식별 가능한 외부 충격이 없습니다.
> 시장 변화는 내부 구조 요인 중심으로 해석하며,
> MSS와 Track A 중심의 분석을 유지합니다."

### Example 2 — L2 Moderate Shock
> "중간 규모의 외부 이벤트가 시장에 단기 충격을 가하고 있습니다.
> 현재는 일시적 반응 가능성이 있으나,
> Credit 시장으로의 전염 여부를 즉시 확인합니다.
> 전염이 확인되지 않으면 변동성 확대 구간으로 해석하며,
> TQQQ 추가 매수는 보류합니다."

### Example 3 — L3 Major Shock (전염 미확인)
> "대형 외부 충격이 시장 전반에 영향을 미치고 있습니다.
> 충격의 규모가 크며 구조 전염 가능성을 배제할 수 없습니다.
> TQQQ를 포함한 레버리지 포지션을 대부분 축소하고,
> Credit 시장의 반응을 3~5거래일 내에 확인합니다.
> 전염이 없으면 충격 소화 후 재진입을 검토합니다."

### Example 4 — L4 Systemic Shock + Track A L3 결합
> "시스템 수준의 외부 충격이 발생하였으며,
> 동시에 Credit → Funding 경로의 전염이 확인되고 있습니다.
> 이 충격은 이미 구조화 단계에 진입한 것으로 판단됩니다.
> 모든 레버리지 자산을 즉각 청산하고 현금/달러 자산으로 전환합니다.
> 충격 소화 및 Transmission 경로 해소 확인 전까지 신규 진입을 금지합니다."
