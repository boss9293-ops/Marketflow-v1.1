# Semiconductor Intelligence Architecture Notes

- Added the `Semiconductor Intelligence` parent section.
- Added top-level tabs for `SOXX/SOXL Lens` and `AI Infrastructure Radar`.
- Kept the existing Semiconductor Lens content inside the SOXX/SOXL Lens tab.
- Added an AI Infrastructure Radar placeholder only; no live radar data or new data pipeline was added.

- Each placeholder theme shows `SOXX Link` as TBD and `Status` as Placeholder.

## Product Structure

- SOXX/SOXL Lens: quantitative SOXX holdings, contribution, residual participation, and SOXL daily sensitivity.
- AI Infrastructure Radar: exploratory AI infrastructure theme monitoring outside direct SOXX attribution.
- SOXX Link Layer: future classification bridge for Direct SOXX, Indirect SOXX, and Outside SOXX themes.

## Guardrails

- SOXX/SOXL Lens remains the quantitative SOXX-based screen.
- AI Infrastructure Radar placeholder data is not presented as live data.
- Radar themes are not direct SOXX contributors unless explicitly classified.
- Historical context only. Not a forecast or trading signal.
